# Yacht Rental Website

Static website for yacht rental. Users can browse different yachts and view details of each yacht. **Note:** The website is static and does not allow actual bookings.

## Technologies
- HTML
- CSS
- Images, small animations

## How to View
Since this is a static website:
1. Clone the repository:
   ```bash
   git clone https://github.com/nomigdalevich/yacht-rental-website.git

2.Navigate to the project folder:
cd yacht-rental-website


3.Open index.html in any web browser to view the website.

Or visit the repository online: [Yacht Rental Website on GitHub](https://github.com/nomigdalevich/yacht-rental-website)
